ICON FILES NEEDED
=================

The extension requires three icon files:
- icon16.png (16x16 pixels)
- icon48.png (48x48 pixels)  
- icon128.png (128x128 pixels)

OPTION 1: Create Simple Icons
You can create simple colored square icons using any image editor:
- Use a blue or green color (#2196F3 or #4CAF50)
- Add text "FA" for "Form Auto-Filler" if desired
- Save as PNG files with the exact names above

OPTION 2: Use Online Icon Generator
Visit https://www.favicon-generator.org/ or similar sites to generate icons

OPTION 3: Skip Icons (Extension will still work)
The extension will function without icons, but Chrome will show a default icon.
You can temporarily comment out the icon references in manifest.json if needed.

OPTION 4: Download Free Icons
Search for "form icon" or "autofill icon" on sites like:
- Flaticon.com
- Icons8.com
- Font Awesome

Make sure to resize them to the required dimensions.
